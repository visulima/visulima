# @visulima/pagination

Flexible pagination utilities supporting both cursor-based and offset-based pagination with metadata generation.

## Key Features

- **Cursor Pagination** - Efficient for large datasets
- **Offset Pagination** - Traditional page-based pagination
- **Metadata** - Total count, page info, has more
- **Framework Agnostic** - Works with any framework
- **TypeScript** - Full type definitions

## Quick Start

```typescript
import { paginate } from "@visulima/pagination";

const result = await paginate(query, {
    page: 1,
    limit: 10,
});

console.log(result.data);
console.log(result.metadata);
```
