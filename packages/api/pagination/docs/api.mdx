# API Reference

## paginate()

Offset-based pagination.

```typescript
function paginate<T>(items: T[], options: { page: number; limit: number }): Promise<PaginationResult<T>>;
```

## cursorPaginate()

Cursor-based pagination.

```typescript
function cursorPaginate<T>(items: T[], options: { cursor?: string; limit: number }): Promise<CursorPaginationResult<T>>;
```

## Types

```typescript
interface PaginationResult<T> {
    data: T[];
    metadata: {
        total: number;
        page: number;
        pages: number;
        hasMore: boolean;
    };
}
```
