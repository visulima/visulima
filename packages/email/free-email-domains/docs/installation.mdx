# Installation

```bash npm2yarn
npm install @visulima/free-email-domains
```

## Import

```typescript
import { isFreeEmail, isFreeDomain, areFreeEmails } from "@visulima/free-email-domains";
```

The raw, sorted domain array is available from the `./domains` subpath:

```typescript
import domains from "@visulima/free-email-domains/domains" with { type: "json" };
```
