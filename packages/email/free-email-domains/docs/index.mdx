# @visulima/free-email-domains

Detect addresses from free email service providers using a comprehensive, regularly-updated list of free email domains.

## Quick Start

```typescript
import { isFreeEmail } from "@visulima/free-email-domains";

isFreeEmail("user@gmail.com");
// true (free provider)

isFreeEmail("ceo@my-company.com");
// false (custom/corporate)
```

## Use Cases

- **Lead Scoring** - Flag free-provider signups for sales qualification
- **Fraud Signals** - Combine with other heuristics for risk scoring
- **B2B Gating** - Require a corporate email for business features
- **Data Quality** - Segment users by email provider type
