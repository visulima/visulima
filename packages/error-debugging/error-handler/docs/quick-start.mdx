# Quick Start

## Express/Connect

```typescript
import express from "express";
import { createErrorHandler } from "@visulima/error-handler";

const app = express();

// Your routes...

// Error handler (must be last)
app.use(
    createErrorHandler({
        debug: true,
    }),
);
```

## Custom Format

```typescript
app.use(
    createErrorHandler({
        debug: true,
        formatError: (error) => ({
            error: error.message,
            code: error.statusCode || 500,
        }),
    }),
);
```
