# @visulima/source-map

Parse, generate, and manipulate source maps for debugging transpiled JavaScript and TypeScript.

## Key Features

- Parse source maps
- Generate source maps
- Map positions
- Support for inline maps

## Quick Start

```typescript
import { SourceMapConsumer } from "@visulima/source-map";

const map = await SourceMapConsumer.fromSourceMap(sourceMap);
const original = map.originalPositionFor({ line: 1, column: 0 });
```
