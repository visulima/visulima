# API Reference

Complete reference for all functions, types, and utilities provided by @visulima/iso-locale.

## Countries

### getByAlpha2

Get country by ISO 3166-1 alpha-2 code.

```typescript
function getByAlpha2(code: string): Country | undefined;
```

**Parameters:**

- `code`: 2-letter country code (e.g., "US")

**Returns:** Country object or undefined

**Example:**

```typescript
getByAlpha2("US"); // { name: "United States", alpha2: "US", ... }
```

---

### getByAlpha3

Get country by ISO 3166-1 alpha-3 code.

```typescript
function getByAlpha3(code: string): Country | undefined;
```

**Parameters:**

- `code`: 3-letter country code (e.g., "USA")

**Returns:** Country object or undefined

---

### getByNumeric

Get country by ISO 3166-1 numeric code.

```typescript
function getByNumeric(code: string | number): Country | undefined;
```

**Parameters:**

- `code`: 3-digit numeric code (e.g., "840" or 840)

**Returns:** Country object or undefined

---

### getCountryByName

Get country by exact name (case-insensitive).

```typescript
function getCountryByName(name: string): Country | undefined;
```

**Parameters:**

- `name`: Country name (e.g., "United States")

**Returns:** Country object or undefined

---

### searchCountries

Search countries by partial name (case-insensitive).

```typescript
function searchCountries(query: string): Country[];
```

**Parameters:**

- `query`: Search query (e.g., "united")

**Returns:** Array of matching Country objects

---

### getEmoji

Get flag emoji for a country.

```typescript
function getEmoji(countryCode: string | number): string | undefined;
```

**Parameters:**

- `countryCode`: ISO 3166-1 alpha-2, alpha-3, or numeric code

**Returns:** Flag emoji or undefined

---

### getCallingCode

Get first country calling code (phone prefix).

```typescript
function getCallingCode(countryCode: string | number): string | undefined;
```

**Parameters:**

- `countryCode`: ISO 3166-1 alpha-2, alpha-3, or numeric code

**Returns:** First calling code (e.g., "+1") or undefined

---

### getCallingCodes

Get all country calling codes.

```typescript
function getCallingCodes(countryCode: string | number): string[];
```

**Returns:** Array of calling codes

---

### getLanguages

Get languages for a country.

```typescript
function getLanguages(countryCode: string | number): string[];
```

**Returns:** Array of ISO 639-3 language codes

---

### getIOC

Get International Olympic Committee code.

```typescript
function getIOC(countryCode: string | number): string | undefined;
```

**Returns:** IOC code (e.g., "USA") or undefined

---

### isValidCountry

Validate country code.


## Locale & BCP 47

### getCurrency

Get currency code from locale or country code.

```typescript
function getCurrency(locale: string): string | undefined;
```

**Parameters:**

- `locale`: Locale string (BCP 47, underscore format, or country code)

**Returns:** ISO 4217 currency code or undefined

**Supports:**

- BCP 47 tags: `"en-US"`, `"pt-BR"`
- Underscore format: `"en_US"`, `"pt_BR"`
- Country codes: `"US"`, `"GB"`

---

### getLocales

Get all country codes (alpha-2) that use a specific currency.

```typescript
function getLocales(currencyCode: string): string[];
```

**Parameters:**

- `currencyCode`: ISO 4217 currency code (e.g., "EUR")

**Returns:** Array of ISO 3166-1 alpha-2 country codes

---

### parseBCP47Tag

Parse a BCP 47 language tag into its components.

```typescript
function parseBCP47Tag(tag: string): { language: string; country?: string; script?: string } | undefined;
```

**Parameters:**

- `tag`: BCP 47 language tag (e.g., "en-US", "zh-Hant-TW")

**Returns:** Parsed components or undefined if invalid

---

### generateBCP47Tag

Generate a BCP 47 language tag from language and country codes.

```typescript
function generateBCP47Tag(languageCode: string, countryCode: string, script?: string): string;
```

**Parameters:**

- `languageCode`: ISO 639-1 language code (e.g., "en")
- `countryCode`: ISO 3166-1 alpha-2 country code (e.g., "US")
- `script`: Optional script code (e.g., "Latn", "Hant")

**Returns:** BCP 47 tag (e.g., "en-US", "zh-Hant-TW")

---

### getBCP47Tags

Get all BCP 47 language tags for a country.

```typescript
function getBCP47Tags(countryCode: string): string[];
```

**Parameters:**

- `countryCode`: ISO 3166-1 alpha-2 country code

**Returns:** Array of BCP 47 tags (e.g., ["en-US"])

---

### isValidBCP47Tag

Validate a BCP 47 language tag format.

```typescript
function isValidBCP47Tag(tag: string): boolean;
```

**Returns:** true if format is valid, false otherwise

---

### iso6393To6391

Convert ISO 639-3 (3-letter) code to ISO 639-1 (2-letter) code.

```typescript
function iso6393To6391(iso6393: string): string | undefined;
```

**Parameters:**

- `iso6393`: ISO 639-3 language code (e.g., "eng")

**Returns:** ISO 639-1 code (e.g., "en") or undefined if not found

---

### convert6393To6391

Get all ISO 639-1 codes from ISO 639-3 codes.

```typescript
function convert6393To6391(iso6393Codes: string[]): string[];
```

**Parameters:**

- `iso6393Codes`: Array of ISO 639-3 codes

**Returns:** Array of ISO 639-1 codes (filtered to only valid mappings)
