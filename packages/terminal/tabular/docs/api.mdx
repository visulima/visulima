# API Reference

## Functions

### `createTable(options?)`

Create a new table instance.

### `createGrid(options?)`

Create a new grid instance.

## Table Methods

### `setHeaders(headers)`

Set table headers.

### `addRow(row)`

Add a single row.

### `addRows(rows)`

Add multiple rows.

### `render()`

Render the table as a string.

## Options

```typescript
interface TableOptions {
    maxWidth?: number;
    showHeader?: boolean;
    wordWrap?: boolean;
    style?: {
        paddingLeft?: number;
        paddingRight?: number;
        borderStyle?: string;
    };
}
```
