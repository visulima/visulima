## Basic Usage

### Simple Spinner

```typescript
import { Spinner } from '@visulima/spinner';

const spinner = new Spinner({
  spinner: 'dots',
  text: 'Processing...'
});

spinner.start();

// Do async work
await processData();

spinner.succeed('Completed!');
```

### With Error Handling

```typescript
const spinner = new Spinner({
  spinner: 'line',
  text: 'Connecting to database...'
});

spinner.start();

try {
  await connectDB();
  spinner.succeed('Connected!');
} catch (error) {
  spinner.fail(`Failed: ${error.message}`);
}
```

### Custom Symbols

```typescript
const spinner = new Spinner({
  spinner: 'breathe',
  text: 'Loading',
  successSymbol: '✓',
  failureSymbol: '✗',
  warningSymbol: '⚠'
});

spinner.start();
await load();
spinner.succeed();
```

## Available Spinners

Get all available spinner names:

```typescript
import { getSpinnerNames } from '@visulima/spinner';

const names = getSpinnerNames();
console.log(names); // ['aesthetic', 'arc', 'arrow', ...]
```

Check out the [Spinner Reference](/docs/spinner/spinners) to see all 101 spinners with previews and details.

## Environment Detection

The spinner automatically detects whether it's running in a TTY (terminal) environment:

```typescript
// Spinner only animates in TTY environments
const spinner = new Spinner({
  spinner: 'dots',
  text: 'Loading...',
  isEnabled: true // Set to false to disable in CI/non-TTY
});
```

## Common Patterns

**Disable in CI/testing:**
```typescript
const spinner = new Spinner({
  spinner: 'dots',
  text: 'Loading...',
  isEnabled: process.stdout.isTTY // Only in terminals
});
```

**Check current status:**
```typescript
const status = spinner.getStatus();
if (status === 'spinning') {
  spinner.setText('Still working...');
}
```
