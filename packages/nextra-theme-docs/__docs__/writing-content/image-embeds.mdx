## Image

### Using Markdown

The [markdown syntax](https://www.markdownguide.org/basic-syntax/#images) lets you add images using the following code

```md
![title](/path/image.jpg)
```

### Using Embeds

To get more customizability with images, you can also use [embeds](/writing-content/embed) to add images

```html
<img height="200" src="/path/image.jpg" />
```

Nextra supports [HTML tags in Markdown](https://www.markdownguide.org/basic-syntax/#html). This is helpful if you prefer HTML tags to Markdown syntax, and lets you create documentation with infinite flexibility.

### Videos

For YouTube videos use:

```html
<iframe
    width="560"
    height="315"
    src="https://www.youtube.com/embed/lJIrF4YjHfQ"
    title="YouTube video player"
    frameborder="0"
    allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
    allowfullscreen
/>
```

For other videos, use:

```html
<video controls className="w-full aspect-video" src="link-to-your-video.com"></video>
```

### iFrames

Loads another HTML page within the document.

```html
<iframe src="https://www.youtube.com/embed/lJIrF4YjHfQ"></iframe>
```

To have videos autoplay, add `autoplay = {true}`.

<Callout type={"tip"}>You can also upload your videos to storage services like Cloudinary to reference them in your Nextra documentation!</Callout>

### Line Breaks

Add a linebreak

```html
<br />
```

<Callout type={"tip"}>

Although not required, we recommend adding the `alt` and `title` attributes to images for better SEO and accessability. Learn more at [image SEO](https://yoast.com/image-seo-alt-tag-and-title-tag-optimization/).

</Callout>
