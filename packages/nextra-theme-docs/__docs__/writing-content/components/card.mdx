## API

The `Cards` component takes the following props:

### `title`

The title of the card

-   Type: `string`
-   Required: `true`

### `icon` (optional)

The icon of the card

-   Type: `ReactNode`

### `href` (optional)

The url that clicking on the card would navigate the user to

-   Type: `string`

### `children` (optional)

The content of the card

-   Type: `ReactNode`
