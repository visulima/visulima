You can add any number of tabs.

<Tabs>
    <Tab title="pnpm">**pnpm**: Fast, disk space efficient package manager.</Tab>
    <Tab title="npm" disabled>
        **npm** is a package manager for the JavaScript programming language.
    </Tab>
    <Tab title="yarn">**Yarn** is a software packaging system.</Tab>
</Tabs>

```mdx
import { Tab, Tabs } from "@visulima/nextra-theme-docs/components";

    <Tab title="npm" disabled>
        **npm** is a package manager for the JavaScript programming language.
    </Tab>
    <Tab title="yarn">**Yarn** is a software packaging system.</Tab>

```

### Default Selected Index

You can use the `defaultIndex` prop to set the default tab index:

```mdx
import { Tab, Tabs } from "@visulima/nextra-theme-docs/components";

<Tabs defaultIndex="1">...</Tabs>
```

And you will have `npm` as the default tab:

<Tabs defaultIndex="1">
    <Tab title="pnpm">**pnpm**: Fast, disk space efficient package manager.</Tab>
    <Tab title="npm">**npm** is a package manager for the JavaScript programming language.</Tab>
    <Tab title="yarn">**Yarn** is a software packaging system.</Tab>
</Tabs>

## API

The `Tabs` component takes the following props:

### `defaultIndex`

The index of the tab that should be selected by default.

-   Type: `number`
-   Default: `0`

### `selectedIndex`

The index of the tab that should be selected.

-   Type: `number`
-   Default: `undefined`

### `children`

The content of the tabs.

-   Type: `React.ReactNode | React.ReactNode[]` -> needs to be an array of `Tab` components
-   Required: `true`

### `onChange`

A callback function that is invoked when the selected tab changes.

-   Type: `(index: number) => void`
-   Default: `undefined`

### `prefix`

The prefix of the tabs head key.

-   Type: `string`
-   Default: `tabs`

The `Tab` component takes the following props:

### `title`

The title of the tab. Short titles are easier to navigate.

-   Type: `string | React.ReactNode`
-   Required: `true`

### `children`

The content of the tab.

-   Type: `React.ReactNode`
-   Required: `true`

### `disabled`

Whether the tab is disabled.

-   Type: `boolean`
-   Default: `false`

### `storageKey`

The key used to store the tab state in local storage.

-   Type: `string`
-   Default: `undefined`

### `disableScrollBar`

Whether the tab should disable the scroll bar.

-   Type: `boolean`
-   Default: `false`

### `classes`

The classes used to style the tab and tabs (root).

-   Type: `object`
    -   tab: `string`
    -   tabs: `string`
-   Default: `undefined`
