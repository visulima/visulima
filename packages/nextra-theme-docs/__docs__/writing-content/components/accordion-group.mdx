    </Accordion>

</AccordionGroup>

## API

The `AccordionGroup` component takes the following props:

### `children`

List of Accordions.

-   Type: `Accordion`
-   Required: `true`


### `styleType` (optional)

You can choose between `rounded` and `flushed` style.

-   Type: `rounded | flushed`
-   Default: `rounded`
