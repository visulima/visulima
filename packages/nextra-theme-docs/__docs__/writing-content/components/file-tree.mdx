## API

The `FileTree` component takes the following props:

none

The `FileTree.Folder` component takes the following props:

### `name`

The name of the folder

-   Type: `string`
-   Required: `true`

### `defaultOpen` (optional)

Whether the folder should be open by default

-   Type: `boolean`
-   Default: `false`

The `FileTree.File` component takes the following props:

### `name`

The name of the file

-   Type: `string`
-   Required: `true`
