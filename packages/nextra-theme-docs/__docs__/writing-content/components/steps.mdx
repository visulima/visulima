# Step Component

A built-in component of the Docs Theme to turn a numbered list into a visual
representation of steps.

## Example

<Steps>
### This is the first step

This is the first step description.

### This is the second step

This is the second step description.

### This is the third step

This is the third step description.

</Steps>

## Usage

Wrap a set of markdown h3 headings with the `Steps` component to turn them into
visual steps.

<Steps>

### Step 1

Content for step 1.

### Step 2

Contents for step 2.

</Steps>

```mdx
import { Steps } from "@visulima/nextra-theme-docs/components";

<Steps>

### Step 1

Contents for step 1.

### Step 2

Contents for step 2.

</Steps>
```

Or to use the component without a title, use the `noTitle` prop.

<Steps noTitle>

Content for step 1.

Contents for step 2.

</Steps>

```mdx
import { Steps } from "@visulima/nextra-theme-docs/components";

<Steps noTitle>

Contents for step 1.

Contents for step 2.

</Steps>
```

## API

The `Steps` component takes the following props:

### `noTitle` (optional)

The type of the Callout.

-   Type: `boolean`
-   Default: `'false'`
