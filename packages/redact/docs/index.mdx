# Visulima Redact

Visulima Redact is a high-performance library for redacting sensitive information in objects, strings, and arrays. Built on top of [compromise](https://github.com/spencermountain/compromise) and [dot-prop](https://github.com/sindresorhus/dot-prop), it provides a comprehensive solution for secure logging and public data output.

## Features

- **Fast and Efficient**: Optimized for performance with minimal overhead
- **Comprehensive Redaction**: Handles objects, strings, arrays, Errors, Maps, and Sets
- **Deep Copy**: Does not modify input objects, always returns a new copy
- **Circular Reference Handling**: Safely handles circular references in objects
- **Smart Detection**: Automatically filters valid JSON strings and URL query parameters
- **Built-in Rules**: Includes 40+ default rules for common sensitive data patterns
- **NLP-Powered**: Anonymize text using Natural Language Processing for categories like emails, phone numbers, organizations, and people
- **Customizable**: Define your own redaction rules and patterns
- **TypeScript Support**: Full TypeScript type definitions included
- **Framework Agnostic**: Works in Node.js and browser environments

## Quick Example

```typescript
import { redact } from '@visulima/redact';

const data = {
  user: {
    email: 'user@example.com',
    password: 'secret123',
    creditCard: '4532-1234-5678-9010'
  },
  apiKey: 'sk_live_1234567890abcdef'
};

const redacted = redact(data, ['password', 'user.email', 'apiKey']);

console.log(redacted);
// {
//   user: {
//     email: '<USER.EMAIL>',
//     password: '<PASSWORD>',
//     creditCard: '4532-1234-5678-9010'
//   },
//   apiKey: '<APIKEY>'
// }
```
