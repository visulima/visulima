## Start to use it

<Steps>
### Install

In your project directory, run the following command to install the dependency:

<Tabs>
    <Tab title="pnpm">``` pnpm i @visulima/error ```</Tab>
    <Tab title="npm">``` npm i @visulima/error ```</Tab>
    <Tab title="yarn">``` yarn add @visulima/error ```</Tab>
</Tabs>

### Ready to Go!

> In this Example we will extend the VisulimaError class to create a custom error

```ts
import { VisulimaError } from "@visulima/error";

class CustomError extends VisulimaError {
    public constructor(message: string) {
        super({
            name: "CustomError"
            message
        });
    }
}

throw new CustomError("This is a custom error");
```
