# @visulima/disposable-email-domains

Validate and block disposable/temporary email addresses using a comprehensive, regularly-updated list of disposable email domains.

## Quick Start

```typescript
import { isDisposableEmail } from "@visulima/disposable-email-domains";

isDisposableEmail("user@mailinator.com");
// true (disposable)

isDisposableEmail("user@gmail.com");
// false (legitimate)
```

## Use Cases

- **User Registration** - Block temporary emails
- **Email Validation** - Ensure real email addresses
- **Spam Prevention** - Reduce fake accounts
- **Data Quality** - Maintain clean user database
