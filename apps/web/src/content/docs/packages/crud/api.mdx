# API Reference

## PrismaAdapter

Create adapter for Prisma client.

```typescript
new PrismaAdapter({ prismaClient: prisma })
```

## nodeHandler()

Create Next.js API route handler.

```typescript
function nodeHandler(adapter): Promise<Handler>
```

## Query Parameters

- `filter[field]` - Filter by field value
- `sort` - Sort fields (prefix `-` for descending)
- `page` - Page number
- `limit` - Items per page
- `include` - Include relations
