# Installation

```bash npm2yarn
npm install @visulima/crud @prisma/client prisma
```

## Setup Prisma

```bash
npx prisma init
npx prisma migrate dev
```

## Import

```typescript
import { PrismaAdapter } from "@visulima/crud";
import { nodeHandler } from "@visulima/crud/next";
```
