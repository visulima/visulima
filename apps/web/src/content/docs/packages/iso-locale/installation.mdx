# Installation

```bash npm2yarn
npm install @visulima/iso-locale
```

## Import

```typescript
import { getByAlpha2, getEmoji, getCurrency } from "@visulima/iso-locale";
```

### Subpath Imports

```typescript
import { getByAlpha2 } from "@visulima/iso-locale/countries";
import { getByCode } from "@visulima/iso-locale/currencies";
import { getTimezonesByCountry } from "@visulima/iso-locale/timezones";
```

## Requirements

- **Node.js:** 22.21 or higher
