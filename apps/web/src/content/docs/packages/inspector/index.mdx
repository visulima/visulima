# @visulima/inspector

Advanced object inspection with customizable formatting, circular reference handling, and depth control.

## Key Features

- Deep object inspection
- Circular reference detection
- Custom formatting
- Syntax highlighting
- Depth control

## Quick Start

```typescript
import { inspect } from "@visulima/inspector";

const obj = { name: "John", age: 30 };
console.log(inspect(obj));
```
